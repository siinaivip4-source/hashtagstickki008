# Auto Batch Hashtag Generator Tool

This tool allows you to automate the generation of hashtags for media files (Gif, Image). It features batch processing, auto-detection of media types, and CSV export capabilities.

## Option A: Running Locally (On your PC)

**Prerequisites:** You must have Python (3.8 or newer) installed on your computer.

### Windows (Quick Start)
1. Ensure all files (`app.py`, `requirements.txt`, `run.bat`) are in the same folder.
2. Double-click the `run.bat` file.
3. The script will automatically install dependencies and open the tool in your default web browser.

### Mac/Linux (Manual Start)
1. Open your Terminal and navigate to the project folder.
2. Install the required dependencies:
